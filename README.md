## ⚙️ Instalação e Configuração

Siga os passos abaixo para configurar o ambiente de desenvolvimento e executar o projeto localmente.

### 1. Pré-requisitos

Certifique-se de que você tem o **Python 3** (versão 3.14 ou superior) instalado em sua máquina.

### 2. Configuração do Ambiente Virtual

O projeto foi criado em um jupyter notebook(`.ipynb`), podendo ser importado no google colab ou localmente por qualquer IDE de sua escolha. Caso deseje rodar localmente, é uma prática recomendada usar um ambiente virtual (`venv`) para isolar as dependências do projeto do sistema operacional.

#### **Criação do Ambiente**

- Execute o comando abaixo no terminal, na pasta raiz do projeto:

```bash
python -m venv venv
```

- Depois rode o comando abaixo para insalta os pacotes recomendados:

```bash
source <caminho-do-venv>/bin/activate
```

- Após o venv ser ativado, rode este ultimo comando:

```bash
pip install -r requirements.txt
```

**Obs**: Aconselhamos configurar na sua IDE particular o ambiente python, associado as configurações necessarias para rodar o plugin do **jupyter**, assim conseguindo executar o notebook da forma correta.
